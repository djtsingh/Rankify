# Rankify

**SEO Made Simple, Results Made Clear**

A modern SEO analysis platform built on Azure serverless architecture.

## Tech Stack

- **Frontend**: Next.js 14 (Static Export) on Azure Static Web Apps
- **Backend**: Azure Functions (Node.js 22, TypeScript)
- **Database**: PostgreSQL Flexible Server
- **Package Manager**: pnpm workspaces

## Development

```bash
# Install dependencies
pnpm install

# Run development servers
pnpm dev

# Build for production
pnpm build
```
